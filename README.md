# Voyage Clothing App

A premium, state-of-the-art e-commerce Flutter application built for the **Voyage** clothing brand. This app features a highly modern, minimalist black-and-white aesthetic designed to "wow" users with an immersive shopping experience.

## Features

- **Splash Screen**: Sleek black entry screen with the elegant Voyage serif logo.
- **Authentication**: Modern Login and Sign Up flows with floating bottom sheet designs and social login buttons (Google & Apple).
- **Home Screen**: Dynamic product listing featuring categories, a sleek search bar, and trending items using local image assets.
- **Product Details**: Immersive full-screen image views with a pull-up "bottom sheet" containing sizes, quantities, and KOKO/Mintpay installment options.
- **Wishlist**: Save your favorite items with a clear layout displaying product info, prices, and direct "Add to Cart" actions.
- **Cart & Checkout**: Full checkout flow with seamless subtotal calculations, shipping fees, and an elegant success modal.
- **Profile Management**: Sleek dark-themed profile header with layered avatar and a unified clean menu card for easy navigation.

## Design Aesthetic
- **Color Palette**: High-contrast Black, White, and subtle Greys.
- **Typography**: Clean, modern fonts combined with elegant serif touches (Times New Roman) for the main logo to achieve a premium fashion feel.
- **UI Patterns**: Generous padding, rounded corners, subtle shadows, floating navigation bars, and bottom sheet overlays.

## Getting Started

To run this project locally:

1. Clone this repository.
2. Ensure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
3. Fetch the dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```

## Assets

This project utilizes local assets for product imagery. To add new products, place the `.png` files into the `assets/images/` directory and ensure they are referenced correctly in the UI code.
